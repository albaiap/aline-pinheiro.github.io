Farol SOC — Monitoramento RTS In Station
Apps Script SeaTalk Webhook Google Sheets
▶
Dashboard automático de ordens abertas RTS com aging buckets e distribuição de relatórios via SeaTalk e e-mail para múltiplos SOCs no Brasil.
  • Filtragem por operação própria (FBS stations: Franco da Rocha, Jaboatão dos Guararapes, Hidrolândia)
  • Cálculo de aging buckets: 0–2d, 2–5d, 5–10d, 10–15d e +15 dias com índice de criticidade
  • Action Tracker kanban com persistência via Apps Script
  • Envio automático de relatórios HTML por SeaTalk webhook e Gmail
  • Triggers diários às 9h, 15h e 18h com abertura via openById para contexto Web App
  • Lida com mais de 265 mil linhas via importação em chunks com retomada por trigger
